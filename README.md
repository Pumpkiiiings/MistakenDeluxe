## 📄 Descripción

Mistaken es un plugin que trata de recrear los juegos: Dead by Daylight, Forsaken (Roblox)
Bear Alpha (Roblox) y Die Of Death (Roblox), trayendolos a Minecraft.

El plugin fue construido en Kotlin y Java usando tomando de base el plugin https://github.com/PulseBeat02/MurderRun hecho por
**PulseBeat02** para la **1.21.10**.

---


## 🏗️ Dependencias

Para que **NextMistaken** funcione correctamente, el servidor debe contar con:
* **LuckPerms:** Control de rangos y permisos de asesino.
* **Vault:** Economía y recompensas por partida.

**DEPENDENCIAS OPCIONALES**
* **CraftEngine:** Soporte para mecánicas extendidas.
* **PumpkinEffect:** Efectos de muerte custom.
---

## 🛠️ Tecnologías y APIs

* **📄 [Paper API](https://papermc.io/):** El motor principal del servidor, aprovechando las últimas optimizaciones de Java 21.
* **📦 [PacketEvents](https://github.com/retrooper/packetevents):** Motor de gestión de paquetes para un manejo de red a bajo nivel y sin lag.
* **✨ [Advanced Slime Paper (ASP)](https://github.com/InfernalSuite/AdvancedSlimePaper):** Sistema de carga de mundos ultrarrápido diseñado para instancias de minijuegos.
* **🎞️ [Triumph GUI](https://github.com/TriumphTeam/triumph-gui):** Framework de interfaces (GUI) que permite nuestra optimización de **Hybrid Cache**.
* **🔐 [HikariCP](https://github.com/brettwooldridge/HikariCP):** El pool de conexiones SQL más rápido del mundo para el manejo de estadísticas sin tirones.
* **💵 [Vault API](https://github.com/MilkBowl/VaultAPI):** Integración estándar para el sistema de economía de la tienda.
* **⚙️ [PlaceholderAPI](https://github.com/PlaceholderAPI/PlaceholderAPI):** Soporte completo para variables externas en scoreboards y menús.
* **📦 [CraftEngine Support](https://github.com/Xiao-MoMi/craft-engine):** Soporte integrado para mecánicas y modelos avanzados.
